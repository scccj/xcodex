# Godot 回合流程页面示例

这是一个可运行的 Godot 4.x 2D 示例页面，演示了你描述的回合流程：

1. 选择下一步操作（进攻 / 防御 / 蓄力）
2. 点击出牌
3. 再打出
4. 结算
5. 进入下一轮

页面会显示：
- 当前轮次
- 当前状态提示
- 已选择的操作
- 最近操作记录

## 运行方式

1. 打开 Godot，点击 **Import**。
2. 选择本目录下的 `project.godot`。
3. 导入后点击 **Run Project**。

启动后会加载 `scenes/Main.tscn`。
运行后会在控制台打印两回合演示：

1. 回合 1：双方都出云
2. 回合 2：玩家出小波、对手出小防（小波被挡）

你可以在 `scripts/main.gd` 里直接替换卡牌组合，验证任意对局。
