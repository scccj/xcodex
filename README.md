# 云波防御规则（Godot 4.x 最小逻辑实现）

这是一个最小可运行的 Godot 2D 项目，已按你给的规则实现“同时出牌、同时结算”的核心逻辑。

## 已实现规则

- 双方初始：`1` 血、`0` 云。
- 卡牌与费用：
  - 云：`+1 云`
  - 小波：`-1 云`
  - 大波：`-3 云`
  - 小防：`0 云`，只能防小波
  - 大防：`-2 云`，可防小波和大波
- 同时出牌并结算：
  - 小波 vs 小防/大防：被挡
  - 大波 vs 大防：被挡
  - 大波 vs 小防：小防无效，命中则击杀
  - 任一波命中未防守目标：目标死亡（1 血归零）

核心代码在：`scripts/main.gd`。

## 运行方式

1. 打开 Godot，点击 **Import**。
2. 选择本目录下的 `project.godot`。
3. 导入后点击 **Run Project**。

运行后会在控制台打印两回合演示：

1. 回合 1：双方都出云
2. 回合 2：玩家出小波、对手出小防（小波被挡）

你可以在 `scripts/main.gd` 里直接替换卡牌组合，验证任意对局。
